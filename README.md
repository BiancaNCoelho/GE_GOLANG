# Gaussian Elimination - GO

Gaussian Elimination implemented with GO language. 

## Compile and Run

Compile:

>	make sequential

>	make concurrent

>	make parallel
	
Run:

>	./GOsequential [matrixSize] [Seed]

>	./GOconcurrent [matrixSize] [Seed]

>	./GOparallel [matrixSize] [Seed]
